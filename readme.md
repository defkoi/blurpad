# image formatter

## flags

| :--: | :--------: | :--------------------------: |
|  -i  | input.png  |            input             |
|  -o  |     -      |            output            |
|  -p  |   target   | padding (css\|ratio\|target) |
|  -r  | 4:5 1:1.91 |       threshold ratio        |
|  -t  | instagram  |            target            |
